Project Name: Coffee Shop Locator
Timeframe: December 2017



Visit Site

Leveraging APIs for the Sake of Coffee

For this project, I was tasked to create an interactive experience using dynamic data from multiple APIs. I decided to map out every place that sells coffee in The Triangle using the Google Maps API. I then used the Google Places API to populate info windows containing ratings, reviews and contact information about each place.


Through this project I was blown away by the power of APIs. For example, my intention was to map out every spot that sells coffee within a 30-mile radius of NC's Research Triangle. After I finished the project I realized the radius function I wrote was broken, so the online app actually maps out and provides information on every place that sells coffee in the WORLD. I was so amazed that I decided to leave it that way. So go ahead, visit the site and see if you can find your favorite coffee shop near you!
